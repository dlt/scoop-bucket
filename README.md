# Scoop Bucket

Scoop bucket for [pg_glimpse](https://github.com/dlt/pg_glimpse) - a terminal-based PostgreSQL monitoring tool.

## Installation

```powershell
scoop bucket add dlt https://github.com/dlt/scoop-bucket
scoop install pg_glimpse
```

## Updating

```powershell
scoop update pg_glimpse
```
